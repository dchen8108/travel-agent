# Planning

This directory holds product and technical planning docs for the initial `travel-agent` scope.

Current docs:

- `v0-product-spec.md`: Product boundaries, user flow, screen requirements, and notification behavior.
- `v0-wireframes.md`: Low-fidelity screen layouts and card states.
- `v0-data-model.md`: Minimal local-file schema for a personal, local-first build.
- `v0-tech-spec.md`: Proposed local stack, architecture, routes, jobs, and Google Flights email-ingestion design.
- `google-flights-poc.md`: Current feasibility findings, what is proven, and what access is still needed from the user.
- `implementation-plan.md`: Concrete build order and MVP quality gates for the first usable version.
